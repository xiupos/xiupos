| --- | --- | --- |

| ---- | ----- | ----- |

## 名詞類

### 名詞

#### 単数

| | I | IIm ; IIn | IIIь |
| --- | :-: | :-: | :-: |
| Nom | а | - ; о | ь |
| Gen | ы | а | и |
| Dat | е | у | и |
| Akk | у | * ; о | ь |
| Inst | ой | ом | ью |
| Präpos | е | е | е |

```txt
|        | I   | IIm ; IIn | IIIь  |
| ------ | --- | --------- | ----- |
| Nom    | a   | -   ; o   | ь     |
| Gen    | y   | a         |    i  |
| Dat    | e   | u         |    i  |
| Akk    | u   | *   ; o   | ь     |
| Inst   | o j | o m       | ь ju  |
| Präpos | e   | e         |    e  |
```

#### 複数

| | I | IImъ ; IInъ | IIь, IIIь |
| --- | :-: | :-: | :-: |
| Nom | ы | ы ; е | и |
| Gen | - | об ; - | ей |
| Dat | ам | ам | ям |
| Akk | * | * ; е | * |
| Inst | ами | ами | ями |
| Präpos | ах | ах | ях |

```txt
|        | I    | IImъ ; IInъ | IIь IIIь |
| ------ | ---- | ----------- | -------- |
| Nom    | y    | y    ; e    |  i       |
| Gen    | -    | o b  ; -    |  e j     |
| Dat    | a m                  ja m     |
| Akk    | *           ; e       *       |
| Inst   | a mi                 ja mi    |
| Präpos | a x                  ja x     |
```

### 形容詞

| Kurz | а | о | ы |

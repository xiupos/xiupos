[^10]: 執筆時点で10年10ヶ月なので, どちらかというと11年の方が近い. 遅刻しました.

## PHP + HTML (2015/2 ~ 2015/3)

中学受験が終わって一息ついた小学6年生の2月, このブログを始めた. 最初は, 適当に見つけてきた HTML テンプレートを無料レンタルサーバー[^webcrow]でホストしただけ簡素なのものであったが, [Bootstrap](https://getbootstrap.jp/) で記事一覧の見た目を整えたり, PHP で記事本文以外のパーツを共通化してみたりして, 自分なりに「ブログ」の体裁に近付けようとしていた.

[^webcrow]: [ウェブクロウ](https://web.archive.org/web/20150201052857/http://www.webcrow.jp/)というネットオウル(現スターサーバー)の無料レンタルサーバーを使っていた. 次の WordPress も同じ系列の[WPblog](https://web.archive.org/web/20150423035508/http://www.wpblog.jp/)という無料ホスティングサービスである. どちらも2025年時点ではサービス終了している.

具体的な記事の記録が残っていないが, たしか「ブログはじめました」というタイトルで, レンタルサーバーへの HTML テンプレートのアップロード方法か何かを書いた覚えがある. 自己紹介なども律儀に書いていて, 実は今よりもブログっぽかったのかもしれない.

そもそもどうしてブログを始めたか, これはよく覚えていない. ただ, 幼い頃から既にインターネットが普及していたのと, 電子工作少年であったことから, 情報源としての個人ブログの存在はかなり身近なものであった. 個人ブログを読みながら見よう見まねで工作をしていたのだから, 自分も個人ブログで発信したいと思ったのは自然な発想だと思う. 加えて, クラスで流行りの言葉を載せたHTMLを学校のPC室で表示したり, [デュラララ!!](https://www.durarara.com/1st/)にハマって友人グループ用のチャットサイト[^chat]を作ったりしていたので, それらの「工作」の一環としてブログも作ってみようとしたのだと思う.

[^chat]: ネットで拾った PHP + Ajax 構成の掲示板のコードを元に, デザインをデュラララ!!のチャット風にしたり. リアルタイム性を求めてバックエンドに Firebase を使ったり, 小学生なりに改良して使っていた. 中学に進学して友人たちとバラバラになったあともしばらく使われており, まだ LINE がそこまで普及していなかったこともあって, 貴重な連絡手段になっていた.

ちなみに, しばらく自己紹介に「文章力を上げるためのブログ」と書いていたが, ご覧の通り, 10年続けてもまるで効果はなかった.

## WordPress (2015/3 ~ 2015/5)

素の HTML で記事を書くのに1ヶ月も耐えられず, WordPress に移行した. ようやく積極的に記事を書くようになっていたが, いかんせん幼いので「〇〇を買った。これから遊んでいきたい」(以後それに関する記事は無し) のような記事ばかりであった気がする. WordPress を使っていたのはたった2ヶ月であるが, 次の livedoor ブログに記事を引き継いだので, 本格的にブログが始動したのはこの時期と言っていいだろう.

## livedoor ブログ (2015/5 ~ 2017/9)

5月に livedoor ブログが「[完全無料化](https://staff.livedoor.blog/archives/51883193.html)」したというニュースを見て, 現金な中坊であった自分はまんまと登録してしまった. 商業的なブログサービスであるから, WP と比べて自由度は格段に減ったが, 元々自由を享受するほどの技術力がまだ無かったので, 当時の自分はむしろ「ブログ」らしいサイトになって喜んでいた.

この頃は月1程度で記事を書いていた. 特に印象に残っているのは, 情報があまりないソフトウェアのインストール方法を書いた記事である. ある掲示板にてそれを読んでインストールできたという報告があって, 自分の記事が誰かの参考になったことがとても嬉しかった[^sanko]. また, [中学校の自由研究](../../electronics/hs-summer/index.md)で電子工作をする機会がより増えたのもあって, そういった記事もそれなりに書いていた.

[^sanko]: 以後も度々そういった機会があって, その都度喜んでいる. 元々は自己満足なブログであるが, 飽き性な自分が10年も続けられたのは, 自分もお世話になった個人ブログ文化への貢献の気持ちが大きい.

## はてなブログ (2017/9 ~ 2020/4)

高校受験を前に[^juken], 前々から livedoor ブログのスマホでの表示に不満があったのも重なって, 同じようなブログサービスであるはてなブログに移行した. また, この頃からサイトデザインにもこだわり始めるようになった.

[^juken]: 自分の人生を思い返してみると, 受験の直前ほど新しい趣味を始める傾向にあるらしい. 今考えるとヒヤヒヤする.

![2018年頃, はてなブログ時代のサイトトップ](_1.png)

画像は2018年頃のサイトトップである. ブログタイトルも名義も今と違っているのがわかる. またブログサービス時代はタグ・カテゴリーや検索機能があって, 今より閲覧者に寄り添ったブログだったと言えよう.

この時期以降の記事は, ほとんどが今のブログにまで引き継がれている (例えば画像中の記事は[これ](../../camera/night-sky/index.md)). 逆に言えば, livedoor ブログ以前の記事はしばらく後に消してしまった. 小中学生が書いた稚拙な記事だったとは言え, どこかに残しておいてもよかったと後悔している.

## Hugo (2020/4 ~ 2023/4)

コロナ禍で時間ができて, GitHub Actions も覚えたことから, 商業的なブログサービスをやめてSSGを使ったブログにすることを決意した.

![2022年頃, Hugo時代のサイトトップ](_2.png)

画像は2022年頃のサイトトップである. この頃には名義もブログタイトルも今と同じものである. このブログは [Hugo](https://gohugo.io/) を用いて作成されており, テーマは [vaga/hugo-theme-m10c](https://github.com/vaga/hugo-theme-m10c) を使っていた. この頃から「タグ・カテゴリは廃止」「検索はいらない」とミニマル志向になっていき, また記事での敬体を常体に改めたことから, いよいよ閲覧者に気を使わないブログになってきた.

## Astro (2023/4 ~)

3年間 Hugo を使っていたが, 一から自分でテーマを作成したいと常々思っており, 大学2年生になって学部に移行し, 時間もできたことから, 重い腰を上げてテーマ作成をした. ついでに Fediverse の友人に教えてもらった [Astro](https://astro.build/) で全体を作り直した.

![2025年頃, Astro時代のサイトトップ](_3.png)

画像は執筆時点 (2025年12月) でのブログのトップである. 投稿年月日とタイトルだけを羅列した目次は, ミニマルを目指したからなのか, 実装が面倒だったからなのか. どちらにしろ, 年数の割に記事の少ないこのブログには十分の機能性だと思う.

以上が10年の思い出であるが, 今後も続けていくだろうと思う. またキリのいい周年で振り返り記事を書けたら嬉しい. 続くといいな.
